## Kenapa Menggunakan Tool Ini?

Berbeda dengan translator biasa, tool ini dibuat khusus untuk konversi aksara dengan dukungan lengkap:

- **Sandangan** (tanda vokal seperti wulu, suku, taling)
- **Pasangan** (konsonan mati yang ditulis di bawah huruf)
- **Aksara Murda** (huruf kapital untuk nama dan gelar)
- **Aksara Swara** (huruf vokal mandiri)
